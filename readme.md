# 🎧 Fake Video Player Prank

Ein kleiner HTML/CSS/JS Web-Prank mit einem Fake-Musik-/Video-Player.

Das Projekt sieht aus wie ein normaler Video-Player:
- ▶️ Video starten
- 🔇 Mute-Button
- 🎬 normale Player-Optik

Aber nach mehreren Klicks auf den Mute-Button passiert ein versteckter Effekt.

---

## ✨ Features

- 🎥 Eigenes Video einbindbar
- 🔇 Fake-Mute-Button
- 💀 Überraschungseffekt
- 🔊 Separater Soundeffekt (`scare.mp3`)
- 📱 Funktioniert im Browser
- ⚡ Einfach nur HTML, CSS & JavaScript
